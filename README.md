Municipales 2014 : carto de l'abstention
===============

## Dependencies
# Node
```
wget http://nodejs.org/dist/v0.10.26/node-v0.10.26.tar.gz
tar -xzvf node-v0.10.26.tar.gz
cd node-v0.10.26
./configure
make
sudo make install
```
# GDAL
```
sudo apt-get install gdal-bin
```

## Install
```
sudo npm install -g bower
bower install
make all
```

open index.html
